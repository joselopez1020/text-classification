# text-classification
Modelo de scikit-learn para la clasificación de reseñas

Clasificación de Sentimientos en Texto

En este portafolio crearemos un modelo que nos ayudara a resolver un problema importante, saber si el producto, servicio, contenido, etc, es del agrado de las personas mediante los resultados del modelo

Para crear el modelo utilizamos:

Python · Pandas · NumPy · scikit-learn · Matplotlib · Seaborn · os · random · shape

El dataset utilizado en el modelo se encuentra en Kaggle

Link: https://www.kaggle.com/datasets/pradeeshprabhakar/preprocessed-dataset-sentiment-analysis
